Key Research Projects

Nonnegative Independent Component Analysis

I am developing a new algorithm for neuroscience calcium imaging data with Dr. Dmitri Chklovskii and Kushal Kolar at Flatiron institue. This algorithm aims to overcome limitations in existing analysis packages (e.g. CaImAn). The new algorithm has shown promising results in identifying signal components in high-dimensional noisy data, resolving source overlaps in space and time, without requiring constraints on source morphological and temporal shapes.
